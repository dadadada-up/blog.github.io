使用GitHub Pages和Hexo搭建个人博客完全指南

在这篇文章中，我将分享如何使用GitHub Pages和Hexo框架搭建一个专业的个人博客网站。这个方案完全免费，而且可以自定义程度高，适合程序员和技术爱好者使用。

为什么选择GitHub Pages + Hexo?

  • 完全免费：GitHub Pages提供免费托管
  • 无需服务器：静态网站无需维护服务器
  • 版本控制：基于Git的工作流，方便管理内容
  • Markdown支持：使用Markdown编写文章，专注于内容
  • 主题丰富：Hexo有大量精美主题可选
  • 插件生态：丰富的插件扩展功能

准备工作

在开始之前，你需要准备以下工具和账号：

  1. GitHub账号：用于创建仓库和托管网站
  2. Git：版本控制工具
  3. Node.js：Hexo基于Node.js
  4. 文本编辑器：推荐VS Code或Sublime Text

步骤九：添加功能插件

根据需要，你可以添加各种功能插件，例如：

  • 搜索功能：npm install hexo-generator-searchdb --save
  • 站点地图：npm install hexo-generator-sitemap --save
  • RSS订阅：npm install hexo-generator-feed --save
  • 文章统计：npm install hexo-symbols-count-time --save

步骤十：自定义域名（可选）

如果你有自己的域名，可以设置自定义域名：

  1. 在你的DNS提供商处添加CNAME记录，指向你的用户名.github.io
  2. 在Hexo项目的source目录下创建一个名为CNAME的文件，内容为你的域名
  3. 重新部署博客

总结

通过以上步骤，你已经成功搭建了一个基于GitHub Pages和Hexo的个人博客。这个博客完全免费，而且可以高度自定义。随着你对Hexo的深入了解，你可以添加更多功能，使你的博客更加专业和个性化。
